Program is simple to use 

Starts by asking user to enter a number between 10-100, if the user inputs a number out of that range, it will keep asking until it receives the right range. 

After shuffling, it'll ask the user to guess if the next number is Higher or lower, user has to type 'H' or 'L'. It'll keep asking until it receives the appropriate input. 

When the game is over, it will display final score.
